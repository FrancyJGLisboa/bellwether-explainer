How to talk to Bellwether.

Bellwether is a queryable archive of 12 U.S. and global farm-data sources — WASDE, USDA export sales, CFTC positioning, crop condition, drought and ENSO — stored exactly as published and handed back with the source file and release date on every number.

You talk to it through an AI assistant (Claude, or any MCP-capable agent) that has Bellwether attached as a tool. This page is the two things people ask first: what to ask it, and how to connect it. It is fundamentals only — it holds no price data of any kind, and says so when you ask.

It answers fundamentals, and refuses the rest by name.

The fastest way to get a useless answer is to ask for a price. Bellwether has none — and rather than guess, it tells you which tool does.

Ask it about — the archive holds this

  • Balance sheets — ending stocks, production, use, stocks-to-use (WASDE, PSD)
  • Export sales & shipments, by commodity and destination (USDA ESR, GATS)
  • Speculative positioning — managed-money longs/shorts (CFTC COT)
  • Crop condition, progress and acreage (USDA NASS, FSA)
  • Drought coverage, ENSO phase, teleconnection indices (NDMC, CPC, NOAA)
  • Country crop narratives from USDA attachés (GAIN reports)

It will refuse — and defer, not bluff

  • “December corn futures price?” — no price data of any kind; defers the price leg
  • “Cash basis at the Gulf?” — no cash, basis or term structure
  • “Will it rain in Mato Grosso next week?” — no weather forecasts
  • “What's the trade consensus for the next WASDE?” — no consensus or whisper numbers

A question with a price leg and a fundamentals leg is half-answerable: it delivers the in-corpus half, cited, and names the missing half rather than inventing it.

Two habits that make answers better

Ask for the change, not just the level (“…and how did it move?”) — it computes the delta rather than eyeballing it. And for anything historical, name the date (“the August 2020 WASDE”) so it reads the archive as it stood then, not as later revisions rewrote it.

Two doors — one is open today.

Bellwether is an MCP server. Any MCP-capable agent can attach it. How you connect depends on whether your assistant can launch a local program.

Local — works today

For assistants that run on your machine and can start a subprocess: Claude Code, Claude Desktop, Cursor, Copilot CLI. Clone the repo, then:

uv sync
uv run bellwether all --since 2020 --root archive
uv run bellwether facts && uv run bellwether fts
claude mcp add bellwether -- \
  uv run bellwether serve --root archive

Now any session in that folder answers from the archive, cited. Full steps are in § 08 of the explainer.

Remote — not live yet

Web assistants (Claude.ai, ChatGPT) can't launch a local program — they need a hosted URL. That endpoint (bellwether serve-http, bearer-token auth) is built and staged, but not currently deployed, so there is no public link to point at today.

Want hosted access, or a walk-through connecting the local path? Ping Francy — that's the honest route until the endpoint goes live.

One caveat that decides whether to trust the answer

Retrieval works with any model, but the discipline that makes the answer trustworthy — cite every number, respect the date, refuse what's out of corpus — was only validated on Claude Sonnet or better. Smaller models drop citations and blur the boundary. Attach it wherever you like; for an answer you'd act on, run it behind Sonnet-or-stronger.

What a good answer looks like.

Once it's attached, paste one question and watch what comes back. A real answer carries its receipts — if it doesn't, an internal gate vetoes it before you see it. The exchange below shows the shape of an answer (illustrative figures) — run it live for the real, dated numbers.

You:  Latest WASDE US corn ending stocks, and the change from last month?

It:   2026/27 US corn ending stocks are 1,960 million bushels
      (wasde/2026/wasde_2026-07.md, released 2026-07-11) — up 3
      from 1,957 the prior month (wasde/2026/wasde_2026-06.md).

Every figure names the file and date it came from, and the change is computed, not asserted. Ask a price question next and watch it decline by name — that refusal is the feature, not a gap.
